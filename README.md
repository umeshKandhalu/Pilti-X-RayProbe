# Clinical Decision Support System

A mobile application for Chest X-ray analysis and clinical decision support, built with Flutter and FastAPI.

## Project Structure
- `app/`: Flutter Mobile Application
- `backend/`: Python FastAPI Backend
- `install_dependencies.sh`: Helper script to install Homebrew and Flutter
- `INSTRUCTIONS.md`: **READ THIS FIRST** - Detailed setup and usage guide.

## Quick Start
Permission and environment setup is required before running.
Please refer to [INSTRUCTIONS.md](INSTRUCTIONS.md) for step-by-step guidance.
